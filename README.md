| [Kubernetes Default Namespaces & Core Components](kubernetes/kubernetes-default-namespaces-and-core-components.md) | The namespaces and core components every cluster ships with — `kube-system`, `kube-public`, `kube-node-lease`, CoreDNS, kube-proxy, and kubeadm bootstrap. |

| [Kubernetes Control Plane Generic Guide](kubernetes/kubernetes-control-plane-generic-guide.md) | Common issues and fixes for `kube-apiserver`, `etcd`, `kube-scheduler`, `kube-controller-manager`, and `cloud-controller-manager`. |
| [Kubernetes Control Plane Troubleshooting Guide](kubernetes/kubernetes-control-plane-troubleshooting-guide.md) | Focused on managed clusters (EKS / AKS / GKE) with notes for self-managed (kubeadm) clusters. |
| [Kubernetes Multi-Master (HA) Troubleshooting Guide](kubernetes/kubernetes-multi-master-troubleshooting-guide.md) | Diagnosing and fixing problems specific to high-availability clusters with multiple control-plane nodes — etcd quorum, API server load balancing, leader election, and certificates. |
